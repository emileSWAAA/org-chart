# Contributing

Thanks for contributing to Org Chart.

## Getting Started

1. Use Node.js 20.19+.
2. Install dependencies: `npm install`.
3. Start dev server: `npm run dev`.
4. Run checks before submitting: `npm run lint` and `npm run build`.

## Development Guidelines

- Keep changes focused and small.
- Update docs when behavior or CLI usage changes.
- Do not commit generated build output (`dist/`) or local temp files.
- Preserve existing formatting and naming conventions.

## Pull Requests

1. Create a branch from `main`.
2. Describe what changed and why.
3. Include screenshots for UI updates when relevant.
4. Ensure lint/build pass locally.

## Reporting Issues

Please include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment (OS, Node.js version)
- Sample manifest when possible
